HTML

HTML Basics:

  • HyperText Markup Language (scripting language).
  • This is the markup language that is used to create/design a web page.
  • HTML is responsible for the placement/arrangement of the content on the web page and the formatting of the content on the web page.
  • HTML files have the extension .html or .htm.
  • To create a web page in HTML, we need a text editor like Notepad.
  • To run a web page in HTML, we need a web browser.
  • Note: HTML might work differently on different browsers.
  • HTML is open-source and free. No cost of procurement.

HTML Rules:

  • HTML is case-insensitive. It does not treat upper-case and lower-case alphabets differently. Tags can be written in any case. For example, <body> and <BODY> are the same.
  • HTML is a tag/elements-based language. Tags are ways to give commands or create/place content on a web page. All the tags are predefined; no user-defined tags exist in HTML.
  • HTML does not show any errors, but it might not produce the correct output.

Versions of HTML:

  • Version Year:
  • HTML 1991
  • HTML+ 1993
  • HTML 2.0 1995
  • HTML 3.2 1997
  • HTML 4.01 1999
  • XHTML 2000
  • HTML5 2014 (2012)

Types of Tags:

  • Singular tags
  • Paired tags
  • We do not need to close these. They do not contain any content.
  • Syntax: <tagname /> or <tagname> or <tagname/>
  • For example: <hr />, <hr/>, <hr>
