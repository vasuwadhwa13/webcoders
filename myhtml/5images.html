<img> tag:

  • Image tag: Represents an image on a web page.
  • Singular tag: Used alone, does not require closing tag.
  • Used to display images: Displays images on a web page.
  • <img /> tag attributes:
    • src: Specifies the image file's path or URL.
    • width: Specifies the image's width in pixels.
    • height: Specifies the image's height in pixels.
    • alt: Provides alternative text for screen readers.
    • align: Aligns the image with surrounding content.
    • hspace: Sets horizontal space around the image.
    • vspace: Sets vertical space around the image.
