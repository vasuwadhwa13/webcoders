Heading Tags:

  • 6 heading tags that come with a pre-defined font style (font family, size, color).
  • These tags also come with spacing before and after the heading.
  • These tags also come with auto line break.
  • All these 6 heading tags are paired tags.

Styling Tags:

  • <b>Text</b> used to bolden some text
  • <i>Text</i> used to make text italic
  • <u>Text</u> underlines the text
  • <strong>Text</strong> is bold only
  • <em>Text</em> emphasis - is italic only
  • <sup>Text</sup> superscript
  • <sub>Text</sub> subscript

Attributes of a Tag:

  • Attributes are the extra properties that we can set on a tag.
  • A lot of tags in HTML have attributes.
  • Tag attribute syntax: <tagname attribute_name="value" attribute_name="value">
