Table:

  • Table: Used to show information in a tabular manner.
  • Tags Used:
  • <table></table> - Table container tag
  • <tr></tr> - Table row tag
  • <td></td> - Table data tag / columns
  • <th></th> - Table heading tag
  • <thead></thead> - Table header
  • <tbody></tbody> - Table body
  • <tfoot></tfoot> - Table footer
  • Syntax:
  • <table>
  • </th>
  • <th>Heading</th>
  • </tr>
  • </thead>
  • <tbody>
  • <tr>
  • <td>Content</td>
  • <td>Content</td>
  • </tr>
  • </tbody>
  • <tfoot>
  • <tr>
  • <td>Content</td>
  • <td>Content</td>
  • </tr>
  • </tfoot>
  • </table>
  • Attributes:
  • border: in pixels
  • bordercolor: color of the border
  • cellspacing: distance between cells
  • cellpadding: margins of the cell
  • bgcolor: background color
  • width: in pixels or in % of the screen size
  • height: in pixels or in % of the screen size
  • align: to align the table with respect to the web page
  • Attributes:
  • align: horizontal alignment of the content
  • bgcolor: background color
  • height: in pixels or %
  • width: in pixels or %
