Hyperlink:

  • Creating a navigation link to a target web page or a target file when we click on an entity (text / image / etc.)
  • Types of Hyperlinks:
  • Standard web links
  • External web links
  • Email links
  • Telephone links (works for mobile browsers)
  • Navigating to a specific part of a web page links
  • Tag used for hyperlinking: <a></a>
  • Anchor tag (<a>) is a paired tag
  • Inside the <a></a> tags, we put the text or the image or entity that we want to create a hyperlink.
  • Standard web links: (opening a web page of the same website)
  • <a href="target_web_page.html">Content</a>
  • Note: The unvisited web page links are blue in color and underlined | the visited web pages links are purple in color and underlined
  • Attributes of <a>:
  • ->href: Hyperlink reference. Here we give the target web page
  • ->target: Where will the target web page be opened. _blank ensures that the target web page opens in a new window or a new tab.
  • External web links: (opening some other website)
  • <a href="http://www.xyz.com" target="_blank">Content</a>
  • Email links: (sending email via a configured email application/client)
  • <a href="mailto:emailaddress">Mail Me</a>
