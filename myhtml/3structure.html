Structure of an HTML5 Document

         <!DOCTYPE html>
         <html>
            <head>
               <title>Title of the webpage</title>
            </head>
            <body>
               <!-- Web page content goes here -->
            </body>
         </html>
Explanation:
  • The <head> tag is used to store metadata of the webpage.
  • Metadata refers to data about data, which includes information about the webpage.
  • For example, the <title> tag inside the <head> tag carries the title of the webpage.
  • The <body> tag contains the content of the webpage.
