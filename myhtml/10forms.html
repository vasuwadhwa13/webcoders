Forms:

    Creating forms in HTML:
  • We design forms in HTML to take user input on websites.
  • The entire form is enclosed within the <form></form> paired tag.
  • Attributes of the <form> tag:
    • id: The unique ID of the form. It is not displayed anywhere.
    • action: Specifies the target page where the form data will go when the form is submitted.
    • method: Specifies the HTTP method to be used. Can be "get" or "post".
