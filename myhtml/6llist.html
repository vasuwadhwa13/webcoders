Lists:

  • Types of Lists in HTML:
  • 1. Unordered list
  • 2. Ordered list
  • 3. Definition list

Unordered Lists:

  • Bulleted lists:
  • Tags used: <ul></ul> and <li></li>
  • <ul> unordered list
  • <li> list item
  • <ul> attribute: type=”circle” or “disc” or “square”
  • Syntax: 
                    <ul>
                <li>Item</li>
                <li>Item</li>
                <li>Item</li>
                </ul>

Ordered Lists:

  • Tags used: <ol></ol> and <li></li>
  • <ol> ordered list
  • <li> list item
  • <ol> attribute: type=”1” or “A” or “a” or “I” or “i”
  • start=”give the start position”
  • reversed: Displays the numbering in reverse order
  • Syntax: 
                    <ol>
                <li>Item</li>
                <li>Item</li>
                <li>Item</li>
                </ol>

Definition Lists:

  • This list is used to define some entities.
  • Tags used: <dl></dl> and <dt></dt> and <dd></dd>
  • <dl> definition list
  • <dt> definition term
  • <dd> definition data
  • Syntax: 
                    <dl>
                <dt>Term</dt>
                <dd>Definition</dd>
                </dl>
