Frames:

  • Web Layouts using <iframe>:
  • Tag: <iframe></iframe> represents inline frame.
  • This tag is used to embed another document inside the HTML document.
  • Syntax:
    • <iframe src="target_page_url" title="description" width="Xpx" height="Xpx" name="iframe_name"></iframe>
  • Example:
    •             <!DOCTYPE html>
            <html>
            <head>
            <title>iframe demo</title>
            </head>
            <body>
            <h1>iframe demo</h1>
            <iframe src="fifth.html" title="Amazon" width="500px" height="400px" name="iframe_amazon"></iframe>
            </body>
            </html>
